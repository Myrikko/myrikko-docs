# SDK Reference

Myrikko supports familiar SDK workflows so you can integrate without replacing your existing application structure.

## Choose an SDK

| SDK | Recommended for | Guide |
|---|---|---|
| OpenAI SDK | OpenAI-compatible chat completion workflows | [OpenAI SDK](openai-sdk.md) |
| Anthropic SDK | Anthropic Messages API workflows | [Anthropic SDK](anthropic-sdk.md) |
| Google GenAI SDK | Google GenAI-compatible workflows | [Google GenAI SDK](google-genai-sdk.md) |

## Before you begin

You need:

- A Myrikko account
- A valid Myrikko API key
- Available API credits
- A supported model ID
