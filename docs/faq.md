## 1. Why should I use Myrikko?

Myrikko gives developers a simpler, more reliable, and more cost-effective way to build with Chinese frontier AI models.

With Myrikko, you can:

* **Integrate once** — Access Chinese frontier AI models like Kimi, GLM, Qwen, and DeepSeek through a unified API. Selected global models (GPT, Claude, Gemini) are also available.
* **Evaluate models more easily** — Compare supported models through a consistent API without maintaining separate provider integrations.
* **Improve reliability** — Intelligent routing automatically switches between supported providers when available, helping reduce service interruptions.
* **Monitor everything in one place** — Track usage, costs, billing, and request logs from a unified dashboard.
* **Reduce AI API costs** — Developer-friendly pricing with no platform markup on supported models.

## 2. Which AI models are available on Myrikko?

Myrikko provides access to a growing collection of leading Chinese AI models, including Kimi, GLM, Qwen, DeepSeek, and more.

We also support selected global models such as Claude, GPT, and Gemini, making it easy to evaluate different model families through a single API.

For the latest list of supported models, please visit the Model Marketplace page.

## 4. Which SDKs does Myrikko support?

Myrikko supports multiple popular SDKs, including OpenAI SDK, Anthropic SDK, and Google GenAI SDK.

In most cases, you only need to update your API key, base URL, and model parameter to start sending requests through Myrikko.

Please refer to the Quickstart guide for SDK Integration setup examples.

---

## 5. How do I get started?

Getting started takes just a few minutes:

1. Create a Myrikko account.
2. Verify your email address.
3. Use the default API key or create a new one.
4. Top up your account or use your available credits.
5. Update your base URL and API key.
6. Start sending requests.

---

## 6. Do I need a separate API key for each model?

No.

A single Myrikko API key gives you access to our full range of supported models.

---

## 7. How is pricing calculated?

Pricing is based on token usage.

Each model has its own input and output token pricing, which is displayed on the Models Marketplace page.

You only pay for what you use.

---

## 8. Is there any platform markup?

Myrikko doesn't add platform markup to supported models — you pay the price shown on the platform.

---

## 9. Where can I view my usage and costs?

You can monitor everything from the dashboard, including:

* API requests
* Token usage
* Costs
* Model usage

You can download your usage details from **Settings → Data & Account**.

---

## 10. What happens if a model provider becomes unavailable?

Myrikko continuously monitors provider availability.

If an issue is detected, requests can be automatically routed to another available provider when supported, helping improve service reliability.

---

## 11. How does automatic routing work?

Myrikko intelligently selects an available provider based on factors such as availability, latency, and service health.

This helps improve reliability without requiring changes to your integration.
